Project Information

About this project:

wxSVG is C++ library to create, manipulate and render Scalable Vector Graphics (SVG) files with the wxWidgets toolkit.

Download

You can download wxSVG releases from Sourceforge Download Page. For the last changed please see ChangeLog.

Using of wxSVG

Loading and Displaying of SVG

There are two methods to display SVG:

  • Using wxSVGCtrl widget:
    m_svgCtrl = new wxSVGCtrl(this);
    m_svgCtrl->Load(wxT("tiger.svg"));
  • Using Render method in wxSVGDocument class:
    wxSVGDocument* svgDoc = new wxSVGDocument;
    svgDoc->Load(wxT("tiger.svg"));
    wxImage img = svgDoc->Render();
